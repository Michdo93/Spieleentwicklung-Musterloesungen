# Materialien

Sprites, Sounds und sonstige Assets, die im Buch verwendet werden.
Alle Dateien stammen aus dem finalen
[Ninja-Fight-Projekt](https://github.com/Michdo93/NinjaFight) und
werden in den einzelnen Kapiteln nach und nach eingebunden - die
jeweils benötigten Dateien liegen zusätzlich immer direkt im
`assets`-Ordner des betreffenden Kapitels, damit jedes Kapitel für
sich lauffähig bleibt.

## sprites/

| Datei | Verwendung |
|---|---|
| `hero.png` | Spielfigur (Idle/Walk/Jump/Hit/Kick/Throw/Die/SwordHit), 8x8-Raster, 160x150 px pro Zelle |
| `blue.png`, `green.png`, `red.png`, `white.png` | Gegnertypen, gleiches Raster wie `hero.png` |
| `tiles.png` | Level-Kacheln (Boden, Wasser, Leiter, Feuer, Messer, ...) |
| `background.png` | Hintergrundbild für Menue und Level |

## sounds/

| Datei | Verwendung |
|---|---|
| `Game-Menu.mp3` | Musik im Hauptmenue |
| `Lost-Jungle.mp3` | Musik während eines Levels |
| `Coins.mp3` | Sound-Effekt beim Einsammeln von Items |
| `sword.mp3` | Sound-Effekt beim Schwerthieb |
