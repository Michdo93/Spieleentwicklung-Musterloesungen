## Ausführen

```bash
python3 -m http.server 8000
```

## Was hier passiert

1. **Highscores mit localStorage** - speichern, sortieren, auf Top 10
   begrenzen
2. **Sieg oder Niederlage?** - Zeitablauf ist nur DANN eine Niederlage,
   wenn zusätzlich noch Gegner übrig sind
3. **Kumulative Lebenspunkte** - `+=` statt `=` über mehrere Level
